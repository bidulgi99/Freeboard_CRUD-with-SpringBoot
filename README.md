# Freeboard_CRUD-with-SpringBoot
Spring Boot + Oracle DB를 활용한 작은 웹사이트 만들기(자유게시판 Part)

자유게시판을 담당하였으며, 구현 소스와 템플릿은 freeboard 폴더 하에서 작업하였습니다.
